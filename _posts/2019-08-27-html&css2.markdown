css 구문
====================================

css 구문은 property 파트와 selector로 나누어져 있다.
- property

property-name: value;

- selector

위의 property를 html의 h1에 연결하고 싶다면?

```
h1 {
  property-name: value;
  property-name: value;
  property-name: value;
}
```
ID와 Class와 연결 할 수도 있다.

```
#Id_name {
  property-name: value;
  property-name: value;
  property-name: value;
}

.Class_name{
  property-name: value;
  property-name: value;
  property-name: value;
}

h1.Class_name{
  property-name: value;
  property-name: value;
  property-name: value;
}

selector (id, class, tag name){
  property-name: value;
  property-name: value;
  property-name: value;
}

```
